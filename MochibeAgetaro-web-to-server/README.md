# MochibeAgetaro-web-to-server
webアプリのためのapiサーバのプログラムソースリスト

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Deploys production
```
npm run deploy
```
